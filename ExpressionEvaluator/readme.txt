Assignment Description:

Make Java program ExpressionEvaluator which takes an arbitrary number of args, presumed to represent an arithmetic expression in Reverse Polish Notation, evaluates the expression, and then outputs (only) the result. Note: (a) Your program must employ your own link-based class CustomizedStack, from first principles. (b) The only valid args are ints, plus the operators +, -, x, /, %. (c) All arithmetic is presumed to be integer arithmetic.

Here's a typical invocation (which should output 2): java ExpressionEvaluator 17 4 / 1 3 - -5 x + 3 %.
